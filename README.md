
# 🎯 JobTrack Pro - https://devpvansh001.github.io/JobTrack-Pro/

**JobTrack Pro** is a clean, lightweight, and local-first job application management system. It helps job seekers stay organized by tracking every stage of their career journey—from the initial wishlist to the final offer—without requiring an account or a database.

---

## 🚀 Features

* **Visual Dashboard**: Monitor your progress with real-time statistics for total applications, interviews, and offers.
* **Interactive Analytics**: View your application status distribution through dynamic doughnut and bar charts powered by Chart.js.
* **Local Storage**: Your data stays private and is automatically saved to your browser's local storage.
* **Job Management**: Add, edit, and delete applications with details like company name, role, salary, location, and personalized notes.
* **Search & Filter**: Quickly find specific roles or companies using the search bar or filter by application status.
* **Responsive Design**: A modern, mobile-friendly interface built with a purple gradient theme and the "Inter" font family.

---

## 🛠️ Tech Stack

* **HTML5**: Structure and semantic layout.
* **CSS3**: Custom styling, animations, and responsive grid layouts.
* **JavaScript (ES6+)**: Core logic, DOM manipulation, and data persistence.
* **Chart.js**: External library used for rendering visual data analytics.
* **Google Fonts**: Utilizes the "Inter" font for a professional look.

---

## 📂 File Structure

To run the application, ensure your directory looks like this:

```text
/your-project-folder
│
├── index.html   # The structure of the dashboard and landing page
├── style.css    # The visual design and theme
└── script.js    # The application logic and storage functions

```

---

## 🚦 Getting Started

1. **Clone or Download**: Copy the HTML, CSS, and JS code into their respective files in a single folder.
2. **Open the App**: Double-click `index.html` to open it in any modern web browser.
3. **Start Tracking**: Enter a company name and job role in the "Add New Application" section to begin.

---

## 📝 Note on Data Persistence

This application uses `localStorage`. This means your data is tied to the specific browser and device you are using. If you clear your browser's cache or "Site Data," your tracked jobs will be removed.
